Aurora PCM: JEE Flashcards
A high-performance flashcard application designed to help students master the NCERT syllabus for JEE Main & Advanced.

About
Aurora PCM is a study tool built to facilitate active recall for Physics, Chemistry, and Mathematics. It organizes the entire JEE curriculum into bite-sized concept drills, allowing for focused, efficient study sessions.

Features
Structured Learning: Organized by subject (P, C, M) covering over 80 chapters.

Data-Driven Prep: Over 2,000+ flashcards to test conceptual understanding.

Optimized UI: A dark-themed, responsive interface designed for minimal distraction.

Tech Stack
Framework: React

Deployment: Vercel
